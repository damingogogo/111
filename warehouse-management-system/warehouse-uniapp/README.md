# 物资到货入库管理系统 uni-app 手机端

## 使用方式

1. 用 HBuilderX 打开本目录：`warehouse-uniapp`。
2. 当前 `common/config.js` 已配置线上后端：`http://38.76.214.13:9000`。
3. 后端需要使用本次更新后的 `warehouse-web`，移动端登录接口为 `POST /api/mobile/login`。
4. 默认账号：`admin / admin123`、`keeper / keeper123`、`keeper2 / keeper123`、`keeper3 / keeper123`、`checker / checker123`。

后端数据库密码不写入代码仓库，部署时请通过 `DB_PASSWORD` 环境变量提供。

## 安卓/鸿蒙打包说明

- 当前项目按 Android APK 兼容方案配置，可安装在支持 Android APK 的华为鸿蒙设备上。
- 已开启 HTTP 明文访问、网络权限、存储权限、录音权限，并保留 arm64-v8a/armeabi-v7a 架构。
- 重新打包前建议在 HBuilderX 中执行一次“清理项目/重新运行”，避免旧缓存影响底部菜单和线上地址。
- 如果目标是鸿蒙 NEXT 原生应用，Android APK 不是同一种包，需要单独做鸿蒙原生适配。

## 已迁移功能

- 手机端登录和会话保持
- 工作台统计
- 物资基础信息查询、新增、编辑、删除、Excel导入、模板下载、Excel导出
- 到货登记查询、新增、编辑、删除
- 到货登记按物资编码自动带出名称、型号、单位
- 来源、运单号、供应商、数量、包装、件数、重量等字段录入
- 入库流程：管库员认领、填写验收完成时间、填写上架时间、填写入库单号
- 到货入库进度Excel导出
- 多账号登录，不同管库员登录后可认领并处理自己的入库流程
- 数量和件数语音输入按钮

语音输入当前优先调用 App 端系统语音能力。大模型识别接口属于客户自费部分，预留在 `common/voice.js` 中，后续拿到客户的大模型接口地址和密钥后接入。
