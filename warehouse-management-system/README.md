# 物资到货入库管理系统

一款用于物资到货登记、管库员认领、验收上架、入库归档的桌面软件。
技术栈：Python + ttkbootstrap + MySQL + openpyxl。

## 快速使用

**普通用户**：直接双击 `启动.bat`（会自动选用 dist 中的 exe；没有则回退到 Python 源码）。也可以直接双击 `dist/物资到货入库管理系统.exe`。

**开发/重新打包**：双击 `重新打包.bat`。

## 功能

- **① 物资基础信息**：增删改查物资编码 / 名称 / 型号 / 单位 / 备注；Excel 批量导入（按编码 upsert）；导出模板和全部数据。
- **② 到货登记**：输入物资编码自动联动名称、型号、单位；手动填写供应商、到货数量、件数、重量；支持 Excel 批量导入。
- **③ 入库管理**：按关键字 / 状态 / 供应商 / 管库员 / 日期范围筛选；管库员认领；填写验收完成时间、上架时间、入库单号；状态自动推进 `待认领 → 已认领 → 已验收 → 已上架 → 已入库`，行底色按状态区分。
- **④ 进度导出**：顶部状态徽章实时统计；筛选条件预览；导出到货/入库进度或物资基础数据。

## 数据库

启动时自动连接 MySQL 并创建：

- 库 `warehouse_mgmt`
- 表 `materials`（物资基础信息）
- 表 `arrivals`（到货登记 + 入库管理）

连接信息见 `config.py`，如需切换服务器请直接编辑。

## Excel 字段约定

**物资基础数据** 列头：

| 物资编码* | 名称* | 型号 | 单位 | 备注 |
| --------- | ----- | ---- | ---- | ---- |

**到货登记** 列头：

| 物资编码* | 名称 | 型号 | 单位 | 供应商 | 到货数量 | 件数 | 重量 | 管库员 | 验收完成时间 | 上架时间 | 入库单号 | 状态 | 备注 |
| --------- | ---- | ---- | ---- | ------ | -------- | ---- | ---- | ------ | ------------ | -------- | -------- | ---- | ---- |

带 `*` 的是必填字段。在软件内点击「下载模板」即可获取标准模板。

## 部署（外发版本）

把 `dist/` 整个目录拷贝给最终用户即可（也可以仅拷 `dist/物资到货入库管理系统.exe`，单文件可独立运行）。
首次启动可能比双击稍慢（PyInstaller 单文件会先解压到临时目录）。
